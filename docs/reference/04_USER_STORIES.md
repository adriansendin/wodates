# 5. Historias de usuario

## HU1 — Registro y creación de perfil
**Como** nuevo usuario, **quiero** registrarme y completar mi perfil **para** comenzar a descubrir personas compatibles.  
**Criterios:**  
- Registro con email y contraseña.  
- Guardado de datos personales, edad, género y preferencias.  
- Subida de avatar.  

## HU2 — Matching y feed
**Como** usuario autenticado, **quiero** ver un feed de personas compatibles **para** dar like o pasar.  
**Criterios:**  
- Filtros por rango de edad, género y preferencias.  
- Sistema de like/pass y generación de matches.  

## HU3 — Chat
**Como** usuario con match confirmado, **quiero** poder chatear **para** conversar con mi match.  
**Criterios:**  
- Creación de chat al producirse match.  
- Envío y recepción de mensajes.
